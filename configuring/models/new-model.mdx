---
title: Add Models
description: Learn how to add new models to Quivr.
---

Quivr allows you to add new models to your account. You can add models from the OpenAI API or from your own custom models.


To add a new model, follow these steps:

<Steps>
<Step title="Go to supabase">
    Go to the [Supabase](http://localhost:54323) dashboard.
</Step>
<Step title="Select the table models">
    Select the table `models`.
    ![Supabase tables](/images/supabase-table.png)
</Step>
<Step title="Add a new model">
    Click on the "Add new row" button.
</Step>
<Step title="Fill in the form">
    Fill in the form with the model's name, maximum tokens, and other details.
</Step>
<Step title="Add the API KEY name to use">
    Add the API KEY name to use the model.
</Step>
<Step title="Add the endpoint to use">
    Add the endpoint to use the model. It needs to be OpenAI compatible. 
</Step>
</Steps>

You have successfully added a new model to your account.
