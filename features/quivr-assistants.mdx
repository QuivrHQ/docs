## Generate usable outputs

<Warning>This feature is Entreprise-Only.</Warning>
<Info>
  A Quivr assistant is an AI agent that generates usable outputs from specific
  inputs.
</Info>

## Our Assistants

### Compliance Check

Extracts criteria from a reference document (e.g., a policy or charter) and verifies if a second document adheres to these specified criteria.

### Consistency Check

Ensures that the data or information in one document is accurately replicated in another, guaranteeing consistency across documents.

### Difference Detection

Highlights any differences between two versions of a document, making it easy to identify changes or modifications.
